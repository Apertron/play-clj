(vector-3! object k & options)

Calls a single method on a vector-3

Options
:add^Vector3 vector
Adds the given vector to this vector
:add^float x, ^float y, ^float z
Adds the given vector to this component
:add^float values
Adds the given value to all three components of the vector.
:clamp^float min, ^float max
Clamps this vector's length to given value
:cpy
:crs^Vector3 vector
Sets this vector to the cross product between it and the other vector.
:crs^float x, ^float y, ^float z
Sets this vector to the cross product between it and the other vector.
:div^float value
:div^float vx, ^float vy, ^float vz
:div^Vector3 other
:dot^Vector3 vector
:dot^float x, ^float y, ^float z
Returns the dot product between this and the given vector.
:dst^Vector3 vector
:dst^float x, ^float y, ^float z
:dst2^Vector3 point
Returns the squared distance between this point and the given point
:dst2^float x, ^float y, ^float z
Returns the squared distance between this point and the given point
:epsilon-equals^Vector3 obj, ^float epsilon
Compares this vector with the other vector, using the supplied epsilon for fuzzy equality testing.
:epsilon-equals^float x, ^float y, ^float z, ^float epsilon
Compares this vector with the other vector, using the supplied epsilon for fuzzy equality testing.
:equals^Object obj
:hash-code
:idt^Vector3 vector
:is-unit
:is-unit^float margin
:is-zero
:is-zero^float margin
:len
:len2
:lerp^Vector3 target, ^float alpha
Linearly interpolates between this vector and the target vector by alpha which is in the range [0,1]. The result is stored in this vector.
:limit^float limit
Limits this vector's length to given value
:mul^float value
:mul^Vector3 other
:mul^float vx, ^float vy, ^float vz
:mul^Matrix4 matrix
Left-multiplies the vector by the given matrix, assuming the fourth (w) component of the vector is 1.
:mul^Quaternion quat
Multiplies the vector by the given {@link Quaternion}.
:nor
Normalizes this vector to unit length. Does nothing if it is zero.
:prj^Matrix4 matrix
Multiplies this vector by the given matrix dividing by w, assuming the fourth (w) component of the vector is 1. This is mostly used to project/unproject vectors via a perspective projection matrix.
:rot^Matrix4 matrix
Multiplies this vector by the first three columns of the matrix, essentially only applying rotation and scaling.
:rotate^float degrees, ^float axis-x, ^float axis-y, ^float axis-z
Rotates this vector by the given angle in degrees around the given axis.
:rotate^Vector3 axis, ^float degrees
Rotates this vector by the given angle in degrees around the given axis.
:scale^float scalar-x, ^float scalar-y, ^float scalar-z
:scl^float value
Scales this vector by the given value
:scl^Vector3 other
Scales this vector by the given vector3's values
:scl^float vx, ^float vy, ^float vz
Scales this vector by the given values
:set^float x, ^float y, ^float z
Sets the vector to the given components
:set^Vector3 vector
Sets the components of the given vector
:set^float[] values
Sets the components from the array. The array must have at least 3 elements
:set^Vector2 vector, ^float z
Sets the components of the given vector and z-component
:slerp^Vector3 target, ^float alpha
Spherically interpolates between this vector and the target vector by alpha which is in the range [0,1]. The result is stored in this vector.
:sub^Vector3 a-vec
Subtracts the given vector from this vector
:sub^float x, ^float y, ^float z
Subtracts the other vector from this vector.
:sub^float value
Subtracts the given value from all components of this vector
:tmp
:tmp2
:to-string
