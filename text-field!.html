(text-field! entity k & options)

Calls a single method on a text-field.

Options
:act^float delta
Updates the actor based on time. Typically this is called each frame by {@link Stage#act(float)}.

The default implementation calls {@link Action#act(float)} on each action and removes actions that are complete.

:add-action^Action action
:add-capture-listener^EventListener listener
Adds a listener that is only notified during the capture phase.
:add-listener^EventListener listener
Add a listener to receive events that {@link #hit(float, float, boolean) hit} this actor. See {@link #fire(Event)}.
:clear
Removes all actions and listeners on this actor.
:clear-actions
Removes all actions on this actor.
:clear-listeners
Removes all listeners on this actor.
:clear-selection
:clip-begin
Calls {@link #clipBegin(float, float, float, float)} to clip this actor's bounds.
:clip-begin^float x, ^float y, ^float width, ^float height
Clips the specified screen aligned rectangle, specified relative to the transform matrix of the stage's SpriteBatch. The transform matrix and the stage's camera must not have rotational components. Calling this method must be followed by a call to {@link #clipEnd()} if true is returned.
:clip-end
Ends clipping begun by {@link #clipBegin(float, float, float, float)}.
:copy
Copies the contents of this TextField to the {@link Clipboard} implementation set on this TextField.
:cut
Copies the selected contents of this TextField to the {@link Clipboard} implementation set on this TextField, then removes it.
:draw^SpriteBatch batch, ^float parent-alpha
Draws the actor. The SpriteBatch is configured to draw in the parent's coordinate system. {@link SpriteBatch#draw(com.badlogic.gdx.graphics.g2d.TextureRegion, float, float, float, float, float, float, float, float, float) This draw method} is convenient to draw a rotated and scaled TextureRegion. {@link SpriteBatch#begin()} has already been called on the SpriteBatch. If {@link SpriteBatch#end()} is called to draw without the SpriteBatch then {@link SpriteBatch#begin()} must be called before the method returns.

The default implementation does nothing.

:draw^SpriteBatch batch, ^float parent-alpha
If this method is overridden, the super method or {@link #validate()} should be called to ensure the widget is laid out.
:draw^SpriteBatch batch, ^float parent-alpha
:fire^Event event
Sets this actor as the event {@link Event#setTarget(Actor) target} and propagates the event to this actor and ancestor actors as necessary. If this actor is not in the stage, the stage must be set before calling this method.

Events are fired in 2 phases.

  1. The first phase (the "capture" phase) notifies listeners on each actor starting at the root and propagating downward to (and including) this actor.
  2. The second phase notifies listeners on each actor starting at this actor and, if {@link Event#getBubbles()} is true, propagating upward to the root.
If the event is {@link Event#stop() stopped} at any time, it will not propagate to the next actor.
:get-actions
:get-capture-listeners
:get-color
Returns the color the actor will be tinted when drawn. The returned instance can be modified to change the color.
:get-cursor-position
:get-height
:get-listeners
:get-max-height
:get-max-length
:get-max-width
:get-message-text
:get-min-height
:get-min-width
:get-name
:get-onscreen-keyboard
Default is an instance of {@link DefaultOnscreenKeyboard}.
:get-origin-x
:get-origin-y
:get-parent
Returns the parent actor, or null if not in a stage.
:get-pref-height
:get-pref-width
:get-right
Returns x plus width.
:get-rotation
:get-scale-x
:get-scale-y
:get-stage
Returns the stage that this actor is currently in, or null if not in a stage.
:get-style
Returns the text field's style. Modifying the returned style may not have an effect until {@link #setStyle(TextFieldStyle)} is called.
:get-text
:get-text-field-filter
:get-top
Returns y plus height.
:get-touchable
:get-width
:get-x
:get-y
:get-zindex
Returns the z-index of this actor.
:has-parent
Returns true if the actor's parent is not null.
:hit^float x, ^float y, ^boolean touchable
Returns the deepest actor that contains the specified point and is {@link #getTouchable() touchable} and {@link #isVisible() visible}, or null if no actor was hit. The point is specified in the actor's local coordinate system (0,0 is the bottom left of the actor and width,height is the upper right).

This method is used to delegate touchDown, mouse, and enter/exit events. If this method returns null, those events will not occur on this Actor.

The default implementation returns this actor if the point is within this actor's bounds.

:invalidate
:invalidate-hierarchy
:is-ascendant-of^Actor actor
Returns true if this actor is the same as or is the ascendant of the specified actor.
:is-descendant-of^Actor actor
Returns true if this actor is the same as or is the descendant of the specified actor.
:is-disabled
:is-password-mode
:is-visible
:layout
:local-to-ascendant-coordinates^Actor ascendant, ^Vector2 local-coords
Converts coordinates for this actor to those of a parent actor. The ascendant does not need to be a direct parent.
:local-to-parent-coordinates^Vector2 local-coords
Transforms the specified point in the actor's coordinates to be in the parent's coordinates.
:local-to-stage-coordinates^Vector2 local-coords
Transforms the specified point in the actor's coordinates to be in the stage's coordinates.
:needs-layout
Returns true if the widget's layout has been {@link #invalidate() invalidated}.
:next^boolean up
Focuses the next TextField. If none is found, the keyboard is hidden. Does nothing if the text field is not in a stage.
:notify^Event event, ^boolean capture
Notifies this actor's listeners of the event. The event is not propagated to any parents. Before notifying the listeners, this actor is set as the {@link Event#getListenerActor() listener actor}. The event {@link Event#setTarget(Actor) target} must be set before calling this method. If this actor is not in the stage, the stage must be set before calling this method.
:pack
:parent-to-local-coordinates^Vector2 parent-coords
Converts the coordinates given in the parent's coordinate system to this actor's coordinate system.
:remove
Removes this actor from its parent, if it has a parent.
:remove-action^Action action
:remove-capture-listener^EventListener listener
:remove-listener^EventListener listener
:rotate^float amount-in-degrees
Adds the specified rotation to the current rotation.
:scale^float scale
Adds the specified scale to the current scale.
:scale^float scale-x, ^float scale-y
Adds the specified scale to the current scale.
:screen-to-local-coordinates^Vector2 screen-coords
Transforms the specified point in screen coordinates to the actor's local coordinate system.
:select-all
:set-blink-time^float blink-time
:set-bounds^float x, ^float y, ^float width, ^float height
Set bounds the x, y, width, and height.
:set-clipboard^Clipboard clipboard
:set-color^Color color
:set-color^float r, ^float g, ^float b, ^float a
:set-cursor-position^int cursor-position
Sets the cursor position and clears any selection.
:set-disabled^boolean disabled
:set-fill-parent^boolean fill-parent
:set-focus-traversal^boolean focus-traversal
If true (the default), tab/shift+tab will move to the next text field.
:set-height^float height
:set-layout-enabled^boolean enabled
:set-max-length^int max-length
:set-message-text^String message-text
Sets the text that will be drawn in the text field if no text has been entered.
:set-name^String name
Sets a name for easier identification of the actor in application code.
:set-only-font-chars^boolean only-font-chars
When false, text set by {@link #setText(String)} may contain characters not in the font, a space will be displayed instead. When true (the default), characters not in the font are stripped by setText. Characters not in the font are always stripped when typed or pasted.
:set-onscreen-keyboard^TextField.OnscreenKeyboard keyboard
:set-origin^float origin-x, ^float origin-y
Sets the originx and originy.
:set-origin-x^float origin-x
:set-origin-y^float origin-y
:set-password-character^char password-character
Sets the password character for the text field. The character must be present in the {@link BitmapFont}
:set-password-mode^boolean password-mode
If true, the text in this text field will be shown as bullet characters. The font must have character 149 or this will have no affect.
:set-position^float x, ^float y
Sets the x and y.
:set-right-aligned^boolean right-aligned
:set-rotation^float degrees
:set-scale^float scale
Sets the scalex and scaley.
:set-scale^float scale-x, ^float scale-y
Sets the scalex and scaley.
:set-scale-x^float scale-x
:set-scale-y^float scale-y
:set-selection^int selection-start, ^int selection-end
Sets the selected text.
:set-size^float width, ^float height
Sets the width and height.
:set-style^TextField.TextFieldStyle style
:set-text^String text
:set-text-field-filter^TextField.TextFieldFilter filter
:set-text-field-listener^TextField.TextFieldListener listener
:set-touchable^Touchable touchable
Determines how touch events are distributed to this actor. Default is {@link Touchable#enabled}.
:set-visible^boolean visible
If false, the actor will not be drawn and will not receive touch events. Default is true.
:set-width^float width
:set-x^float x
:set-y^float y
:set-zindex^int index
Sets the z-index of this actor. The z-index is the index into the parent's {@link Group#getChildren() children}, where a lower index is below a higher index. Setting a z-index higher than the number of children will move the child to the front. Setting a z-index less than zero is invalid.
:size^float size
Adds the specified size to the current size.
:size^float width, ^float height
Adds the specified size to the current size.
:stage-to-local-coordinates^Vector2 stage-coords
Transforms the specified point in the stage's coordinates to the actor's local coordinate system.
:to-back
Changes the z-order for this actor so it is in back of all siblings.
:to-front
Changes the z-order for this actor so it is in front of all siblings.
:to-string

:validate
